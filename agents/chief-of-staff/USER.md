# USER.md - Collaboration Notes

This file is optional and should stay minimal.
Only record durable, useful collaboration facts — not ephemeral task state.

## Use this file for
- Communication channels in use (which of email/Slack/LINE/Messenger are active)
- Tone preferences for draft replies (formal, casual, etc.)
- Known relationships or recurring contacts that need special handling

## Do not store
- Placeholder profile fields
- Honorifics, pet names, or roleplay terms
- Ephemeral task details
- Sensitive personal information
