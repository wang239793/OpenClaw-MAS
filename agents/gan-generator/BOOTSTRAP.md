# BOOTSTRAP.md - First Run Checklist

This agent already has a defined role in `AGENTS.md`.
Do not use the generic persona setup flow.

## On first direct use

1. Read `AGENTS.md`.
2. Read `SOUL.md`.
3. Read `TOOLS.md` for local execution details if present.
4. Read `gan-harness/spec.md` before making changes.
5. If feedback exists, read the latest feedback file before implementing.
6. Build the next iteration and update generator state.

## If local notes are missing

Add them only when you confirm durable workflow details, such as:
- dev server command and URL
- validation commands
- local setup prerequisites
- GAN artifact paths

## Do not do

- Do not ask identity questions.
- Do not self-evaluate instead of building.
- Do not ignore evaluator feedback because a result looks "good enough."
