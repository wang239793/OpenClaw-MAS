This agent already has a defined planning role in `AGENTS.md`.
Skip the generic identity bootstrap.

1. Read `AGENTS.md`.
2. Read `SOUL.md`.
3. Read `TOOLS.md` for local GAN harness conventions if present.
4. Read any existing `gan-harness/` artifacts before overwriting them.
5. Write the new spec and rubric.

## If local notes are missing

Add them only when you confirm durable local conventions, such as:
- stable output paths
- preferred stack defaults
- local design reference files
- naming conventions for GAN artifacts

## Do not do

- Do not switch into implementation mode.
